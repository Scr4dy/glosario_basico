Glosario HTML y CSS

Etiquetas HTML

Las etiquetas HTML son los bloques de construcción de una página web.

<a>

Descripción: Crea hipervínculos a otras páginas, archivos o secciones dentro del documento.

Atributos clave: href, target, download.

Ejemplo:


                <a href="https:google.com" target="_blank">Hola Mundo</a>

                Ir a Google
<body>

Descripción: Contiene todo el contenido visible de la página (textos, imágenes, etc.).

Ejemplo:


                <body>
                    <h1>Hola Mundo</h1>
                </body>
<br>

Descripción: Inserta un salto de línea (sin cerrar como etiqueta contenedora).

Ejemplo:


                Primera línea <br>Segunda línea.

                Primera línea
               Segunda línea.
<button>

Descripción: Crea un botón clickeable (usado en formularios o acciones JS).

Atributos clave: type, onclick.

Ejemplo:


                <button type="button" onclick="alert('Hola'")">Saludar</button>
<div>

Descripción: Contenedor genérico para agrupar elementos y aplicar estilos CSS.

Ejemplo:


                <div class="container">
                    Contenido aquí
                </div>
<form>

Descripción: Define un formulario para enviar datos al servidor.

Atributos clave: action, method, enctype.

Ejemplo:


                <form action="/submit" method="POST">
                    <input type="text" name="nombre" placeholder="Nombre">
                    <button type="submit">Enviar</button>
                </form>
<h1 - h6>

Descripción: Encabezados de diferentes niveles.

Ejemplo:


                <h1>Titulo principal</h1>
                <h1>Subtítulo</h1>

                
Título principal

                
Subtítulo
<head>

Descripción: Contiene metadatos, enlaces a CSS/JS y el título de la página (no visible).

Ejemplo:


                <head>
                    <title>Mi página</title>
                    <link rel="stylesheet" href="styles.css">
                    <meta charset="UTF-8">
                </head>
<html>

Descripción: Raíz del documento HTML. Todos los elementos deben ir dentro de esta etiqueta.

Ejemplo:


                <html>
                    <head>...</head>
                    <body>...</body>
                </html>
<img>

Descripción: Muestra una imagen. ¡Siempre usa el atributo alt!

Atributos clave: src, alt, width, height.

Ejemplo:


                <src="logo.png" alt="Logo de la empresa">
<input>

Descripción: Campo de entrada para datos de usuario (texto, contraseña, checkbox, etc.).

Atributos clave: type, name, placeholder.

Ejemplo:


                <type="email" placeholder="Ingresa tu correo">
<li>

Descripción: Representa un ítem dentro de una lista

Ejemplo:


                <ul>
                    <li>Manzanas</li>
                    <li>Naranjas</li>
                </ul>

                
    
                    
  • Manzanas

  • Naranjas

<link>

Descripción: Enlaza recursos externos como hojas de estilo CSS o favicons.

Atributos clave: rel, href.

Ejemplo:


                <link rel="stylesheet" href="estilos.css">
<p>

Descripción: Define un párrafo de texto.

Ejemplo:


                <p>Este es un párrafo de ejemplo.</p>
<script>

Descripción: Inserta código JavaScript (interno o externo).

Atributos clave: src, async, defer.

Ejemplo:


                <script src="app.js"></script>
<span>

Descripción: Contenedor en línea para aplicar estilos o JS a texto específico.

Ejemplo:


                <p><span class="resaltado">resaltado</span></p>

                
Texto resaltado
<style>

Descripción: Define estilos CSS internos.

Ejemplo:


                <style>
                    body { font-family: Arial; }
                </style>
<table>

Descripción: Crea una tabla con filas (<tr>) y celdas (<td> o <th>).

Ejemplo:


                <table>
                    <tr>
                        <Nombre>
                        <Edad>
                    </tr>
                </table>
<title>

Descripción: Define el título de la pestaña del navegador (importante para SEO).

Ejemplo:


                <title>Mi Sitio Web</title>
<ul>

Descripción: Lista desordenada (con viñetas).

Ejemplo:


                <ul>
                    <Item 1>
                    <Item 2>
                </ul>

Propiedades CSS

Las propiedades CSS son las características que se pueden aplicar a los elementos HTML para definir su apariencia.

Propiedades de texto

Estas propiedades afectan la apariencia del texto en los elementos HTML.

color

Descripción: Color del texto.

Ejemplo:


                p { color: #333; }
font-family

Descripción: Tipo de fuente.

Ejemplo:


                body { font-family: 'Arial', sans-serif; }
font-size

Descripción: Tamaño del texto.

Ejemplo:


                h1 { font-size: 2rem; }
font-weight

Descripción: Grosor (normal, bold).

Ejemplo:


                strong { font-weight: bold; }
text-align

Descripción: Alineación (left, center, right).

Ejemplo:


                div { text-align: center; }

Propiedades de caja

Estas propiedades afectan el modelo de caja de los elementos HTML.

width / height

Descripción: Ancho y alto.

Ejemplo:


                .box { width: 100px; height: 100px; }
margin

Descripción: Espacio exterior.

Ejemplo:


                div { margin: 10px; } /* Arriba/abajo/derecha/izquierda */
padding

Descripción: Espacio interior.

Ejemplo:


                button { padding: 8px 16px; }
border

Descripción: Borde (ancho, estilo, color).

Ejemplo:


                img { border: 1px solid #ddd; }
box-sizing

Descripción: Controla cómo se calcula el tamaño (content-box, border-box).

Ejemplo:


                * { box-sizing: border-box; } /* ¡Recomendado! */

Propiedades de fondo

Estas propiedades afectan el fondo de los elementos HTML.

background-color

Descripción: Color de fondo.

Ejemplo:


                body { background-color: #f0f0f0; }
background-image

Descripción: Imagen de fondo.

Ejemplo:


                header { background-image: url("fondo.jpg"); }

Propiedades de diseño (Layout)

Estas propiedades afectan la disposición de los elementos en la página.

display

Descripción: Tipo de visualización (block, inline, flex, grid).

Ejemplo:


                .container { display: flex; }
position

Descripción: Posicionamiento (relative, absolute, fixed).

Ejemplo:


                .menu { position: fixed; top: 0; }
flexbox

Descripción: Propiedades para crear diseños flexibles.

Ejemplo:


                .container {
                    display: flex;
                    justify-content: center; /* Alineación horizontal */
                    align-items: center;     /* Alineación vertical */
                  }
grid

Descripción: Propiedades para crear diseños de cuadrícula.

Ejemplo:


                .grid-container {
                    display: grid;
                    grid-template-columns: repeat(3, 1fr); /* 3 columnas iguales */
                }

Propiedades de efectos visuales

Estas propiedades afectan la apariencia visual de los elementos HTML.

box-shadow

Descripción: Sombra alrededor de un elemento.

Ejemplo:


                .card { box-shadow: 0 4px 8px rgba(0, 0, 0, 0.2); }
opacity

Descripción: Opacidad (0 a 1).

Ejemplo:


                .overlay { opacity: 0.5; }
transition

Descripción: Transiciones suaves entre estados.

Ejemplo:


                button { transition: background-color 0.3s ease; }

Propiedades de diseño responsivo

Estas propiedades permiten que el diseño se adapte a diferentes tamaños de pantalla.

media queries

Descripción: Aplicar estilos según el tamaño de la pantalla.

Ejemplo:


                @media (max-width: 600px) {
                    body { font-size: 14px; }
                }

Propiedades avanzadas

transform

Descripción: Rotar, escalar o mover elementos.

Ejemplo:


                .icon:hover { transform: rotate(45deg); }
z-index

Descripción: Control de capas (superposición).

Ejemplo:


                .modal { z-index: 1000; }
cursor

Descripción: Cambia el cursor (pointer, not-allowed).

Ejemplo:


                button { cursor: pointer; }
